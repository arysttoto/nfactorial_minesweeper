# nfactorial_minesweeper
Mine Sweeper game made for nfactorial incubator testing))
Install python 3.7
install tkinter, (pip install tk)
run the main.py file))
# INFO
Я написал игру сапёр в которой есть меню, можно закончить игру перезапустить игру или выбрать уровень сложности потом начинается игра. Также можно помечать места где по вашему находяться мины, правым нажатием мыши. человек выигрывает тогда когда он нашел все поля а которых нет мин. по нажатию на кнопку показывается сколько мин вокруг этой кнопки а если кнопка ноль то покажется все ближайший нули и все кнопки с цифрами возле которых мины. для запуска программы нужно установить питон 3.7 и библиотеку tkinter, а далее запустить файл main.py
